# AWS S3 Configuration

## Setup Stepsw

### 1. Install AWS CLI 
```bash
# Download from: https://aws.amazon.com/cli/
# Or via pip:
pip install awscli
```

### 2. Configure AWS Credentials
Choose one method:

**Option A: AWS CLI**
```bash
aws configure
# Enter: Access Key ID, Secret Access Key, Region, Output format please ask anthony for these credentials 
```
